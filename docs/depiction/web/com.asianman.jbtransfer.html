Compatibility: This package is compatible with iOS iOS 11.0 to 14.x.

A tool to help the user switch jailbreaks, devices, or both, all while keeping the sources, packages, and their preferences. I am not responsible for any issues this package may cause. I have tested this on my 8 plus, switching between Taurine and Checkra1n with no issues; however, that may not always be the case, so make a backup before using. Instructions at: https://github.com/Asianman132/JBTransfer
